# Enhancing Agents with Memory

This directory contains supporting source code for the Next '26 Developer Keynote - Demo 3 - Enhancing Agents with Memory. 

`planner_agent/` contains the ADK app that helps users plan marathons while maintaining state across sessions and grounding responses in city rules via AlloyDB.
