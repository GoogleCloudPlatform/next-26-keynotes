# Marathon Plan Review Checklist

## Route Requirements

- [ ] Distance: 26.2 miles (42.195 km) +/- 0.1 miles
- [ ] Start location specified with capacity assessment
- [ ] Finish location specified with capacity assessment
- [ ] Waypoints form a connected, geographically plausible path
- [ ] Elevation profile documented
- [ ] Route surface types documented (road, trail, etc.)

## Logistics Requirements

- [ ] Water stations: minimum 20, every 1-2 miles
- [ ] Nutrition stations: at appropriate intervals (miles 8, 14, 20+)
- [ ] Medical tents: minimum 5, at high-risk areas (hills, sun exposure)
- [ ] Timing system: chip timing with mats at start, finish, and splits
- [ ] Course marshals: minimum 1 per 0.5 miles
- [ ] Start infrastructure: corrals, wave start plan, bag check
- [ ] Finish infrastructure: medals, food, recovery area, results display
- [ ] Porta-potties: adequate count for participant volume

## Safety Requirements

- [ ] Emergency vehicle crossing points every 2 miles
- [ ] Hospital access routes documented (nearest 3 hospitals)
- [ ] Fire station access maintained or detoured
- [ ] Evacuation routes for runners and spectators
- [ ] Crowd management plan with capacity limits
- [ ] Weather contingency (heat, rain, wind thresholds)
- [ ] Communication plan (race control, medical, security)

## Financial Requirements

- [ ] Budget breakdown with cost categories
- [ ] Revenue projections (registration, sponsorship, merchandise)
- [ ] Break-even analysis
- [ ] Insurance and permits accounted for

## Approval Thresholds

| Overall Readiness | Decision |
|-------------------|----------|
| >= 0.85 | Approved |
| 0.60 - 0.84 | Conditional (recommendations provided) |
| < 0.60 | Rejected (blockers must be resolved) |
