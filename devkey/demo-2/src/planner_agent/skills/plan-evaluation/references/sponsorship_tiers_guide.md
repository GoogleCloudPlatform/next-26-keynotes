# Sponsorship Tiers Guide

## Tier Structure

### Title Sponsor (1 slot)
- **Value multiplier**: 2.0× base value
- **Category**: Naming rights
- **Benefits**:
  - Event naming rights (e.g., "Nike Las Vegas Marathon")
  - Logo on ALL materials: bibs, banners, website, social media
  - Exclusive start/finish line branding
  - VIP hospitality suite (20-50 guests)
  - Full social media partnership
  - Press conference participation
  - First right of refusal for subsequent years

### Gold Sponsor (2 slots)
- **Value multiplier**: 0.8× base value
- **Categories**: Sports beverage, athletic footwear
- **Benefits**:
  - Logo on course signage (every 2 miles)
  - Water/nutrition station branding
  - Expo booth premium placement
  - Social media mentions (10+ posts)
  - Product sampling rights

### Silver Sponsor (4 slots)
- **Value multiplier**: 0.4× base value
- **Categories**: Technology, insurance, automotive
- **Benefits**:
  - Logo on event website
  - Standard expo booth
  - Banner placement on course
  - PR mentions in press releases
  - Email newsletter inclusion

### Bronze Sponsor (8 slots)
- **Value multiplier**: 0.15× base value
- **Categories**: Local businesses, restaurants, retail
- **Benefits**:
  - Logo on event website
  - Standard expo booth
  - Single social media mention
  - Goodie bag insert

### In-Kind Sponsor (5 slots)
- **Value multiplier**: 0.1× base value
- **Categories**: Medical supplies, water, food, printing
- **Benefits**:
  - Product placement at relevant stations
  - Logo recognition on website
  - Goodie bag inclusion
  - "Official supplier" designation

## Market Assessment Criteria

| Participant Count | Market | Sponsorship Capture Rate |
|------------------|--------|------------------------|
| >20,000 | Strong | 70-90% of potential |
| 10,000-20,000 | Moderate | 50-70% of potential |
| <10,000 | Developing | 30-50% of potential |
