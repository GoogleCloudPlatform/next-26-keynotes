# Cost Categories

## Fixed Costs (incurred regardless of participant count)

| Category | Base Cost | Notes |
|----------|-----------|-------|
| Permits & City Fees | $25,000 | Varies by city; includes road closure permits |
| Equipment Rental | $50,000 | Barriers, timing systems, stages, PA systems |
| Staffing (Base) | $75,000 | Core team, event director, medical director |
| Marketing (Base) | $30,000 | Website, social media, initial advertising |
| Insurance (Base) | $15,000 | General liability, event cancellation |
| Logistics (Base) | $40,000 | Water stations, porta-potties, waste management |
| Prizes | $25,000 | Prize purse for top finishers |

## Variable Costs (per participant)

| Category | Per Participant | Notes |
|----------|----------------|-------|
| Permits | $0.50 | Additional per-person city fees |
| Equipment | $2.00 | Bibs, timing chips, course materials |
| Staffing | $3.00 | Additional security, medical, volunteers |
| Marketing | $1.00 | Per-runner marketing cost |
| Insurance | $0.80 | Per-participant liability coverage |
| Logistics | $4.00 | Water, nutrition, porta-potties per runner |

## Cost Formula

```
total_cost = Σ(base_cost + per_participant × count)
```

## Cost Reduction Strategies

1. **In-kind sponsorships**: Water, medical supplies, food
2. **Volunteer programs**: Reduce staffing costs by 30-40%
3. **Multi-year permits**: Negotiate discounts for recurring events
4. **Shared equipment**: Partner with other events for timing systems
