# Plan Evaluation Skill

**Goal:** Provide verification data for the event including capacity, budget,
demographics, and nuisance checking.

## Resources

### Scripts
- `scripts/check_evaluation.py`: Evaluates plan against quality thresholds.
- `scripts/project_revenue.py`: Estimates event revenue.
- `scripts/analyze_demographics.py`: Checks participant inclusivity.
